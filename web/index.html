Core Concept

Secure Vaults

BlindKey allows users to create private, password-protected vaults on their device. Each vault is strictly isolated.

  • Each vault protected by a unique password
  • Files accessible only within the app
  • Auto-lock when app is backgrounded
  • Organize: Personal, Work, Shared content
  • Viewer support: Photos, Videos, Documents (PDF)
Controlled Sharing

Share with Confidence

BlindKey introduces the .blindkey format. You decide how recipients use your files.

Granular Permissions

Before sending, decide if the recipient can:

  • Re-share the file
  • Open in external apps
  • Save to device storage

Expiry-Based Access

Set a specific date and time for a shared file to expire. Once expired, access is automatically revoked and the file is deleted.

Allow Extraction Mode

OFF: File is view-only inside BlindKey. No saving, no sharing.
ON: Recipient has full control (download, share, open externally).

Verified Access

Internet is required only when opening shared files with expiry dates. This prevents device time manipulation and ensures rules are enforced. Your personal local vaults work 100% offline.

Privacy First

Strict No-Cloud Policy

BlindKey is built with a zero-knowledge philosophy:

  • Files stored locally on device
  • No cloud sync or backup
  • No password recovery
  • No analytics or data collection

Warning: Passwords cannot be recovered if lost. You are responsible for your keys.
